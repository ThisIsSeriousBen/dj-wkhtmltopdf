.. toctree::

Usage
-----

It contains three models,

    1. HtmlHeaderFooter
        Can manage header and footer and deffirent options of wkhtmltopdf.
    2. HtmlBody
        Can create multiple html bodies.
    3. Html
        Can link up HtmlHeaderFooter and HtmlBody and must configure which view
        should generate pdf.
